This project implements a hybrid lane detection system using OpenCV and Python. The system combines multiple computer vision techniques to accurately detect lane markings under various road and weather conditions.

SAWL-Net

SAWL-Net is an advanced deep learning pipeline for histopathology image classification across multiple microscope magnifications. The model addresses challenges in breast cancer subtype classification through innovative architecture and training strategies.

Key innovations:

  • DenseNet121 backbone with transfer learning
  • Multi-magnification input processing
  • Balanced multi-class sampling techniques
  • Two-stage training for improved robustness

The model demonstrates state-of-the-art performance across 8 cancer subtypes and shows practical applications in medical diagnostics and research.
